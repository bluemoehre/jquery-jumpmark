Installation

Just include the plugin after the jQuery file and it will run automatically on page load.

Chapter 2 will bring you further (down). Click here!

Basic usage

Out of the box the plugin will handle all links and hashes containing the prefix !jump:.
So just place some ID attributes on your pages and link them.
The scrolling will happen when you click on a link pointing to your ID or if you append a corresponding hash to the URL (e.g. by another JavaScript). If you remove the prefix by configuration all links are handled by the plugin.

Go deeper inside

Good to know

There are some special targets #!jump:_top and #!jump:_bottom which will bring you to the top/bottom of the page.
You may add that ID to your page at any place. If present it will be used instead of page's default top.
