Installation

Just include the plugin after the jQuery file and it will run automatically on page load.

Chapter 2 will bring you further (down). Click here!

Basic usage

Out of the box the plugin will handle all links with the scheme #jump:myID.
So just place some ID attributes on your pages and link them.
The scrolling will happen when you click on a link pointing to your ID or if you append a corresponding hash to the URL (e.g. by another JavaScript).

Go deeper inside

Good to know

There is a special key #jump:_top which will bring you back to the top of the page.
You may add that ID to your page at any place. If present it will be used instead of page's default top.

Before you try, you should know you can use your browser's history to get back here - isn't that great?
